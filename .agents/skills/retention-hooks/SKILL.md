## Livra's retention framework

Core loop: Mark → Reflect → Streak → Review → Recommend
Emotional hooks: Progress visibility, streak protection, personal records

When reviewing or proposing any feature, evaluate it against:
1. Does it reduce friction to mark completion?
2. Does it make progress feel visible and real?
3. Does it create a reason to return tomorrow?
4. Does it reward consistency without punishing breaks?
5. Does it have an emotional moment (pride, surprise, encouragement)?

Avoid: gamification for its own sake, notification overload, 
       features that add logging burden without emotional payoff.
